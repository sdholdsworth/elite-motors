# Stylish Vehicle Motor Sales App

Welcome to the Stylish Vehicle Motor Sales App! This application streamlines the process of browsing, selecting, and purchasing vehicles with a modern and user-friendly interface.

## Features

- Browse a wide range of vehicles
- Advanced search and filter options
- Detailed vehicle specifications
- Responsive design for all devices

## Project Goals

- Focus on building fluid responsive wesbite design - the initial start phase of this project will involve mainly the development of the UI for a customer facing website. As it grows, there will be more features and complexity added to facilitate this application into growing into a more full stack application (of a medium to large size)
- Ensure the site is fully accessible for the benefit of all users
- Get to grips with React concepts again and learn new features and concepts of React v19 (latest version at the time of this projects creation)
- Develop meaningful components that will include state management of sorts, that will utilise props, hooks and all concepts of React
- Get to grips with routing with the new React Router v7
- The project will also include Authentication, so user's can create an account and manage things like favourites, vehicle shortlists and manage savable searches
- There will also be a nice theme switch feature added to allow users to convieniently switch between their preffered choice of a light or dark mode
